## 🧠 What I Actually Do

I build interfaces where the **math and the UX have to work together**.

Not just components — rendering pipelines, mesh deformations, real-time canvas sync across 3D scenes. The kind of frontend work where you need to understand geometry to ship something that *feels* instant.

I use **AI tools as a core part of my workflow** — not for boilerplate, but to move faster on hard problems. I write `CLAUDE.md` context files for every codebase so the AI output stays consistent at scale.
